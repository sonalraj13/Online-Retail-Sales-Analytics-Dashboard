An interactive **Tableau dashboard** built to analyze online retail sales data, uncover customer purchasing behavior, and generate actionable business insights using **RFM Analysis**, **Pareto Analysis**, and KPI dashboards.

---

## 📌 Project Overview

This project analyzes online retail transactions to help businesses understand sales performance, customer segments, and revenue trends. It combines data cleaning, customer segmentation, and interactive visualizations to support data-driven decision-making.

---

## 🎯 Objectives

- Analyze sales performance across multiple countries
- Identify top-performing products and markets
- Perform RFM customer segmentation
- Track repeat customer behavior
- Discover revenue trends and business opportunities
- Build an interactive Tableau dashboard for stakeholders

---

## 🛠 Tools & Technologies

- Tableau
- Python (Pandas, NumPy)
- SQL
- Microsoft Excel
- RFM Analysis
- Data Visualization

---

## 📂 Dataset

- **Source:** Online Retail Dataset (UCI Machine Learning Repository)
- **Records:** 1.8M+ retail transactions
- **Countries:** 35+
- **Industry:** E-commerce

---

## 📈 Dashboard Features

- Executive KPI Dashboard
- Revenue & Profit Analysis
- Sales Trend Analysis
- Top Products
- Country-wise Revenue Analysis
- RFM Customer Segmentation
- Pareto (80/20) Analysis
- Price vs Volume Scatter Plot
- Interactive Filters & Parameters

---

## 📊 Key Insights

- Analyzed over **1.8 million retail transactions**
- Customers segmented into **7 RFM groups**
- Best Customers averaged **$7,447** in spending
- Repeat purchase rate was **19%**
- Identified top-performing countries and products
- Highlighted pricing outliers through scatter analysis
